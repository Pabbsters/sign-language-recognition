# BUILD — sign-language-recognition

## Prerequisites
- Modern browser with webcam access (Chrome recommended)
- No installs required — fully client-side

## Run Locally
```bash
cd ~/Projects/sign-language-recognition
# Open index.html in browser, or use a local server:
npx serve .
```
App runs at http://localhost:3000

## Deploy to Vercel
```bash
npm i -g vercel
vercel
```
Follow prompts. App is live in under a minute.
